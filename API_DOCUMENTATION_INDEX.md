## 🚀 Quick Start (5 Minutes)

```bash
# 1. Generate API keys
openssl rand -hex 32
openssl rand -hex 32

# 2. Create .env
cat > .env << 'EOF'
API_KEY_APP1=sk_your_first_key
API_KEY_APP2=sk_your_second_key
DATA_PATH=data
ENV=development
EOF

# 3. Start API
uvicorn api:app --reload

# 4. Test
./test_api_complete.sh
# Or: python test_api_complete.py
# Or: Open http://localhost:8000/api/docs

# ✅ Done!
```
